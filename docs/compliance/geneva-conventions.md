# Genfer Konventionen Compliance

## Übersicht

WILDKATZE-I ist so konzipiert, dass alle Ausgaben mit den Genfer Konventionen konform sind.

## Implementierte Prüfungen

1. **Zivilbevölkerung**: Keine Inhalte, die Zivilisten gefährden
2. **Humanitäres Recht**: Einhaltung des internationalen humanitären Rechts
3. **Kriegsgefangene**: Keine Inhalte, die Gefangene gefährden
4. **Medizinisches Personal**: Schutz von medizinischem Personal

## Automatische Filterung

Das Modell filtert automatisch Inhalte, die gegen folgende Prinzipien verstoßen:

- Unterscheidungsgrundsatz
- Verhältnismäßigkeitsgrundsatz
- Vorsorgeprinzip

## Audit-Trail

Alle generierten Inhalte werden protokolliert für Compliance-Überprüfung.
